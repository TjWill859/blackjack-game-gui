# Blackjack Game GUI

## Overview

Interactive Blackjack game developed using Python and PySide6 with full betting logic, GUI, and custom card graphics.

This project demonstrates object-oriented programming, GUI design, and basic game logic implementation.

## Features

* Fully functional Blackjack game with betting system.
* Custom card graphics for player and dealer hands.
* GUI input for Ace value selection (1 or 11).
* Dealer AI with standard Blackjack rules.
* Handles wins, losses, ties, and Blackjack payouts.
* Out-of-chips detection and game summary window.

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/<your-username>/blackjack-game-gui.git
   ```
2. Navigate to the project folder:

   ```
   cd blackjack-game-gui
   ```
3. Install dependencies (PySide6):

   ```
   pip install PySide6
   ```
4. Run the game:

   ```
   python game_gui.py
   ```
